# TailwindUI Lab

A modern React-based component showcase website built using **Tailwind CSS v4** and **DaisyUI**. This project demonstrates the implementation of multiple responsive UI components with a clean and reusable React component structure.

## Features

* Responsive Navbar
* Hero Section
* DaisyUI Buttons
* Cards Component
* Alerts Component
* Avatar Component
* Forms and Inputs
* Progress Bars
* Table Component
* Countdown Screen
* Responsive Layout
* Reusable React Components

## Technologies Used

* React.js
* Vite
* Tailwind CSS v4
* DaisyUI

## Project Structure


## Installation

```bash
npm install
npm run dev
```
