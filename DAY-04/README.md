# 🛍️ Product Explorer

A simple and responsive React product explorer application built using Fetch API, React Hooks, and React Router.

---

# 🚀 Features

- Fetch products from external API
- Dynamic product cards
- Product search functionality
- Loading spinner
- Responsive layout
- Product details page
- Dynamic routing using React Router
- Clean and professional UI

---

# 🧠 Concepts Used

- React Components
- useState
- useEffect
- Fetch API
- map()
- filter()
- Props
- Conditional Rendering
- React Router DOM
- useParams
- Link Navigation

---

# 🌐 API Used

## All Products

```txt
https://dummyjson.com/products
```

## Single Product

```txt
https://dummyjson.com/products/1
```
