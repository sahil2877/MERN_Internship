# 🌦️ Weather App

A simple and responsive Weather Search App built using React Custom Hooks.

---

## 🚀 Features

- Search weather by city name
- Real-time weather data using API
- Custom Hook implementation
- Loading and error handling
- Responsive modern UI
- Reusable React components

---

## 🛠️ Technologies Used

- React JS
- Vite
- CSS3
- OpenWeatherMap API

---

## 📂 Project Structure

src/
│
├── components/
│ ├── WeatherForm.jsx
│ ├── WeatherCard.jsx
│
├── hooks/
│ ├── useWeather.jsx
│
├── App.jsx
├── main.jsx
├── index.css

---

## ⚡ Installation

```bash
npm install
npm run dev
```

---

## 🔑 API Setup

1. Create account on OpenWeatherMap
2. Generate API Key
3. Replace API key inside:

```js
useWeather.jsx
```
